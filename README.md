# 傾斜割り勘

飲み会のお会計を、参加者ごとの負担度に応じて分けられるシンプルなWebアプリです。

HTML / CSS / JavaScriptのみで動作し、サーバーや外部ライブラリは必要ありません。

## 機能

- 合計金額の入力
- 参加者の追加・削除
- 参加者名の入力
- 3段階の負担度
  - 多め：1.2
  - 普通：1.0
  - 少なめ：0.8
- 原則100円単位での丸め
- 合計金額と一致するよう自動調整
- 計算結果のコピー
- スマートフォン対応

## 使い方

1. `index.html` をブラウザで開きます。
2. 飲み会の合計金額を入力します。
3. 参加者を追加または削除します。
4. 参加者の名前と負担度を選択します。
5. 「割り勘を計算する」を押します。
6. 必要に応じて「結果をコピー」を押し、チャットなどへ貼り付けます。

## 丸めについて

各参加者の支払い金額は、原則として100円単位になるよう調整します。

合計金額が100円で割り切れない場合は、入力された合計金額との完全一致を優先し、1人分の支払いに100円未満の端数を加えます。

## ファイル構成

- `index.html`：画面の構造
- `style.css`：デザインとスマートフォン対応
- `script.js`：参加者操作、割り勘計算、コピー機能
